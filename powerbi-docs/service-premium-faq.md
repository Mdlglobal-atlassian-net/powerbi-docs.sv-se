# <a name="power-bi-premium-faq"></a>Vanliga frågor och svar om Power BI Premium
Bläddra i en lista över vanliga frågor och svar om vad Power BI Premium kan erbjuda.

* Om du har andra frågor [kan du fråga Power BI Community](http://community.powerbi.com/).
* Har du fortfarande problem? Besök [Power BI-supportsidan](https://powerbi.microsoft.com/support/).
* Eller klicka på ikonen **Kontakta mig** ![](media/service-premium-faq/power-bi-contact-me-icon.png)i det nedre högra hörnet i webbläsarfönstret.

## <a name="power-bi-premium"></a>Power BI Premium
**Vad är Power BI Premium?**  
Power BI Premium är ett nytt kapacitetbaserat erbjudande som innehåller

* flexibilitet för att publicera rapporter brett i ett företag, utan krav på att mottagarna har individuella licenser
* större skalning och prestanda från dedikerad kapacitet i Power BI-tjänsten
* möjligheten att ha BI-tillgångar lokalt med Power BI-rapportservern
* en API-yta, en enhetlig uppsättning kapaciteter och åtkomst till de senaste funktionerna för inbäddad analys.

**Vad gör Power BI Premium? Hur fungerar det?**  
Power BI Premium består av kapacitet i Power BI-tjänsten som allokeras exklusivt till varje organisation med stöd av dedikerad maskinvara som hanteras av Microsoft fullt ut. Organisationerna kan välja att använda sin dedikerade kapacitet brett eller att tilldela den till utsedda arbetsytor baserat på antalet användare, arbetsbelastningsbehov eller andra faktorer, och skala upp eller ned eftersom behoven förändras.

**Hur skiljer sig Power BI Pro Power åt från Power BI Premium? Varför krävs Power BI Premium?**  
Power BI Premium är kapacitetsbaserad licens medan Power BI Pro är en användarbaserad licens.

Power BI Premium består av kapacitet i Power BI-tjänsten som exklusivt allokeras till varje organisation med stöd av dedikerad maskinvara som fullt ut hanteras av Microsoft – vilket undanröjer begränsningar rörande datauppsättningens storlek och uppdateringsintervall och ger stöd för isolering på grund av den dedikerade maskinvaran. Power BI Premium ger också flexibiliteten att publicera rapporter brett i ett företag, utan krav på att mottagarna har individuella licenser.

Power BI Pro är till för användare som publicerar rapporter, delar instrumentpaneler, samarbetar med kollegor på arbetsytor och engagerar sig i andra relaterade aktiviteter, som möjligheten att

* redigera och spara anpassade vyer
* skapa personliga instrumentpaneler (fäst på ny instrumentpanel)
* analysera data i Excel eller Power BI Desktop
* dela med stöd för Excel-webbappar
* dela instrumentpaneler och samarbeta med Office 365-grupper
* integrera innehåll med Microsoft Teams.

**Behöver jag Power BI Pro för att använda Power BI Premium?**  
Ja. Power BI Pro krävs för att publicera rapporter, dela instrumentpaneler, samarbeta med kollegor på arbetsytor och engagera sig i andra relaterade aktiviteter.

**Hur mycket kostar Power BI Premium? Hur många SKU:er kommer du att göra tillgängliga?**  
Power BI Premium kan köpas baserat på antalet virtuella kärnor. Priserna anges på sidan [Prissättning för Power BI](https://powerbi.microsoft.com/pricing/). Mer information om noder och v-kärnor finns i [Microsoft Power BI Premium – white paper](https://aka.ms/pbipremiumwhitepaper). Du kan också använda [kalkylatorn](https://powerbi.microsoft.com/calculator/) för att få en uppskattning av hur mycket Premium-kapacitet du kan behöva.

**Vad menas med ”kapacitet”?**  
”Kapacitet” syftar på en namngiven kapacitet som tillhandahålls av en administratör via administratörsportalen för Power BI Premium-kapacitet. En kapacitet är en gruppering av en eller flera noder. Se [Microsoft Power BI Premium – white paper](https://aka.ms/pbipremiumwhitepaper) för mer information.

**Hur faktureras Power BI Premium?**  
Power BI Premium faktureras månadsvis och tecknas på ett år.

**Hur kan jag köpa Power BI Premium?**  
Power BI Premium är tillgängligt som ett tillägg till Power BI Pro i administrationscentret för O365. Mer information finns i [Så här köper du Power BI Premium](service-admin-premium-purchase.md). Du kan också kontakta din Microsoft-representant för mer information.

**Hur vet jag hur mycket Power BI Premium-kapacitet som jag behöver köpa?**  
Använd den här [kalkylatorn](https://powerbi.microsoft.com/calculator/) för att uppskatta hur mycket Power BI Premium-kapacitet du kan behöva.

**När blir Power BI Premium tillgängligt? Kommer det att finnas tillgängligt samtidigt i alla geografiska områden?**  
Power BI Premium kommer att bli allmänt tillgängligt i slutet av andra kvartalet 2017 för alla geografiska områden som för närvarande stöds av Power BI-tjänsten.

**Kommer Power BI Premium att uppfylla kraven på certifiering och regelefterlevnad vid tidpunkten för utgivningen? Hur är det med nationella moln?**  
Power BI Premium kommer att bli allmänt tillgängligt i slutet av andra kvartalet 2017 för alla geografiska områden som för närvarande stöds av Power BI-tjänsten. Nationella moln – specifikt statliga sådana i Tyskland, Kina och USA – planeras att bli tillgängliga under tredje kvartalet 2017. Power BI:s standarder för certifiering och regelefterlevnad kommer att gälla för Power BI Premium då programmet görs allmänt tillgängligt.

**Finns Power BI Premium tillgängligt med Office 365 E5?**  
Power BI Premium är tillgängligt som ett tillägg till Power BI Pro. Office 365 E5 innehåller Power BI Pro. E5-kunder kan köpa Premium som ett tillägg till sina befintliga Pro-licenser.

**Vad är Power BI-rapportservern? Betyder detta att ni gör Power BI tillgängligt lokalt?**  
Power BI-rapportservern är tillgänglig som en del av Power BI Premium och är kompatibel med SQL Server Reporting Services-rapporter, vilket möjliggör distribution av interaktiva Power BI-rapporter och sidnumrerade rapporter lokalt inom gränserna för en organisations brandvägg. Organisationer kan välja Power BI i molnet, eller välja att behålla rapporter lokalt med Power BI-rapportservern och flytta till molnet i egen takt.

Power BI-funktioner som realtids-instrumentpaneler och frågor med naturligt språk och annat i Power BI-tjänsten, är inte tillgängliga som en del av Power BI-rapportservern. 

Mer information om Power BI-rapportservern finns i [Kom igång med Power BI-rapportserver](report-server/get-started.md).

**Är Power BI-rapportservern den funktion som Microsoft har gjort tillgänglig som den tekniska förhandsversionen av Power BI-rapporter för SSRS? Varför ingår detta nu i Power BI när det verkade vara en funktion i SQL Server?**  
Power BI-rapportservern är tillgänglig som en del av Power BI Premium och är kompatibel med SQL Server Reporting Services-rapporter, vilket möjliggör distribution av interaktiva Power BI-rapporter och sidnumrerade rapporter lokalt inom gränserna för en organisations brandvägg.

Med tillkännagivandet av Power BI Premium visar vi för första gången hur den här funktionen kommer att föras till marknaden.

Mer information om Power BI-rapportservern finns i [Kom igång med Power BI-rapportserver](report-server/get-started.md).

**Vad är Power BI-appar?**  
Vi är håller på att utveckla innehållspaket till Power BI-appar för att förbättra hur användare får nya insikter och utforskar dem på företagsnivå. Power BI-apparna är tillgängliga idag och erbjuder ett förenklat sätt för att distribuera instrumentpaneler och rapporter till specifika användare, grupper eller hela företag. Företagsanvändare kan enkelt installera apparna och gå till dem utan problem, de centraliserar innehållet på en enda plats och uppdateras automatiskt. 

Mer information om appar finns i [Vad är appar i Power BI?](service-install-use-apps.md).

**Gör du ändringar i Power BI Pro, Power BI (kostnadsfri), Power BI Desktop et al. som en del av Power BI Premium-distributionen?**  
Nu förenklar vi även skillnaden mellan Power BI Pro och den kostnadsfria tjänsten. Vi har fått feedback om att, trots att den kostnadsfria tjänsten är avsedd för personligt bruk och Power BI Pro möjliggör samarbete, har de funktionella skillnaderna dem emellan skapat förvirring för användarna. Framöver kommer vi att förbättra den kostnadsfria tjänsten så att den får samma funktioner som Power BI Pro, men kommer att begränsa delnings- och samarbetsfunktionerna till enbart Power BI Pro-användare.

Användare av den kostnadsfria Power BI-tjänsten får fördelen av åtkomst till alla datakällor, ökade lagringsgränser för arbetsyta och högre uppdaterings- och strömningshastighet. Ändringarna kommer att gälla från den 1 juni 2017 och du kan läsa mer och detta i [Självregistrering för Power BI](service-self-service-signup-for-power-bi.md). Power BI Desktop fortsätter att vara tillgängligt utan kostnad.

Mer information om den utökade Pro-utvärderingsversionen hittar du under [Aktivering av utökad Pro-utvärderingsversion](service-extended-pro-trial.md).

## <a name="power-bi-free"></a>Power BI (kostnadsfri)
**Vilka funktioner läggs till i den kostnadsfria tjänsten?**  
Från och med den 1 juni 2017 har den kostnadsfria tjänsten funktioner som motsvarar dem i Power BI Pro, förutom vad gäller delning, företagsdistribution och samarbete. Detta inkluderar samma gräns för arbetsytan på 10 GB, upp till 8 schemalagda dagliga uppdateringar för datauppsättningar och en strömningsdatahastighet på maximalt 1 miljon/timme. Vi ger även åtkomst till alla datakällor, inklusive de som är tillgängliga via den lokala datagatewayen.

**Vilka Power BI Pro-funktioner är inte tillgängliga i den kostnadsfria tjänsten?**  
Delning av peer-to-peer-instrumentpaneler, app-arbetsytor (tidigare kallade grupparbetsytor), Analysera i Excel och Power BI-appar (om de inte backas upp av Premium-kapacitet) är funktioner som begränsas till Power BI Pro.

**När kommer de här ändringarna att börja gälla?**  
Ändringarna av den kostnadsfria tjänsten börja gälla den 1 juni 2017.

**Hur kommer berörda användare av den kostnadsfria tjänsten att meddelas om dessa ändringar?**  
Aktiva användare av den kostnadsfria tjänsten som berörs av ändringarna den 1 juni 2017 får ett e-postmeddelande från Microsoft, samt meddelanden när de loggar in i Power BI-tjänsten med början den 3 maj 2017.

Befintliga användare av den kostnadsfria tjänsten som har varit aktiva under det senaste året kan dra fördel av den kostnadsfria 12 månaders utökade utvärderingsversionen av Power BI Pro. Erbjudandet tillhandahåller Power BI Pros kompletta funktioner för att säkerställa att användarna får gott om tid på sig för att anpassa sig till hur man använder tjänsten. Mer information om den utökade Pro-utvärderingsversionen hittar du under [Aktivering av utökad Pro-utvärderingsversion](service-extended-pro-trial.md).

**Förlorar användarna åtkomst till sitt innehåll (instrumentpaneler, rapporter) vid det här tillfället?**  
Nej. De kommer inte att förlora åtkomst till något innehåll som de har överfört till Power BI-tjänsten. Innehåll som delats med andra kanske inte längre är tillgängligt för mottagarna efter den 1 juni 2017. På samma sätt har de kanske inte heller längre åtkomst till innehåll som andra har delat med dem.

Befintliga användare av den kostnadsfria tjänsten som har varit aktiva under det senaste året kan dra fördel av den kostnadsfria 12 månaders utökade utvärderingsversionen av Power BI Pro. Erbjudandet tillhandahåller Power BI Pros kompletta funktioner för att säkerställa att användarna får gott om tid på sig för att anpassa sig till hur man använder tjänsten. Mer information om den utökade Pro-utvärderingsversionen hittar du under [Aktivering av utökad Pro-utvärderingsversion](service-extended-pro-trial.md).

**Hur kan användare veta om de är berättigade till den utökade Power BI Pro-utvärderingsversionen?**  
Se [de allmänna villkoren](https://aka.ms/power-bi-trial) för den utökade Pro-utvärderingsversionen.

Mer information om den utökade Pro-utvärderingsversionen hittar du under [Aktivering av utökad Pro-utvärderingsversion](service-extended-pro-trial.md).

**Vad ska användare göra om de inte är berättigade till den utökade Power BI Pro-utvärderingsversionen?**  
Registrera sig för en 60 dagars Power BI Pro-utvärderingsversion av standardtyp [här](https://app.powerbi.com/signupredirect?pbi_source=web).

**Ändrar introduktionen av Power BI Premium funktionerna i den kostnadsfria tjänsten?**  
Användare av den kostnadsfria tjänsten med åtkomst till dedikerad kapacitet i Power BI Premium har möjligheten att ta emot innehåll som distribueras till dem av Power BI Pro-användare via publicerade appar.

**Vad ändras med Power BI Pro?**

* redigera och spara anpassade vyer
* skapa personliga instrumentpaneler (fäst på ny instrumentpanel)
* analysera data i Excel eller Power BI Desktop
* Dela och visa Excel-arbetsböcker med stöd för Excel-webbappar
* dela instrumentpaneler och samarbeta med Office 365-grupper
* integrera innehåll med Microsoft Teams.

## <a name="power-bi-embedded"></a>Power BI Embedded
**Vad har ändrats i Power BI Embedded?**  
Som en del av Power BI Premium konvergerar vi Power BI Embedded med Power BI-tjänsten för att leverera en enda API-yta, en enhetlig uppsättning funktioner och åtkomst till de senaste funktionerna. Framöver rekommenderar vi dem som är intresserade av att bädda in Power BI i sina appar att börja med Power BI Desktop och gå över till Power BI Premium för distribution.

**Vilka är fördelarna med Power BI Premium när det gäller att bädda in analyser i appar?**  
Power BI Premium är ett kapacitetsbaserat erbjudande som körs på dedikerad maskinvara och hanteras fullt ut av Microsoft som en del av Power BI-tjänsten. Power BI Premium erbjuder nya förmåner som flexibla och förutsägbara kostnader, förbättrad innehållshantering för utvecklare och full tillgänglighet till Power BI:s funktioner för inbäddade scenarier och uppfyller därmed alla behov i din apputvecklingslivscykel.

**Behöver jag migrera min befintliga app som skapats i Power BI Embedded?**  
Power BI Embedded fortsätter att vara tillgängligt för befintliga appar enligt följande schema:

| Licensieringsavtal | Tillgänglighet för Power BI Embedded |
| --- | --- |
| Företagsavtal |Fram till att det befintliga avtalet löper ut |
| Direkt och CSP |Ett år från att Power BI Premium är allmänt tillgängligt |

Ja, du bör börja fundera på att migrera din inbäddade analyslösning till Power BI Premium. Mer information finns i [Så här migrerar du innehåll från Power BI Embedded-arbetsytesamlingar till Power BI](developer/migrate-from-powerbi-embedded.md).

**Hur migrerar jag min befintliga app som skapats i Power BI Embedded?**  
Se [migreringsdokumentationen](developer/migrate-from-powerbi-embedded.md) för anvisningar om hur du migrerar appar.

**Kommer Microsoft inte längre att satsa på Power BI Embedded (kostnadsfri)?**  
För den tidigare modellen erbjöd vi ett begränsat antal sessioner för utveckling och testning. I det nya scenariot har kunderna även fortsatt alternativ tillgängliga för att börja utveckla och testa kostnadsfritt på vägen mot att bli produktionsklara. Kunderna kan dra nytta av en tvåmånaders Power BI Pro-utvärderingsversion för att komma igång med utveckling och testning. Power BI Desktop, datagransknings- och rapportredigeringsverktyget, fortsätter att vara tillgängligt utan kostnad.

**Är Power BI Embedded fortfarande en Azure-tjänst? Var kan jag köpa, tillhandahålla och hantera Power BI Embedded?**  
Power BI Embedded konvergerar med Power BI-tjänsten och kommer därför framöver att köpas, etableras och hanteras i enlighet med detta. Mer information finns i [Inbäddning med Power BI](developer/embedding.md).

## <a name="power-bi-report-server"></a>Power BI-rapportserver
**Varför ingår Power BI-rapportservern i Power BI Premium istället för att säljas som en separat produkt?**  
Med Power BI Premium introduceras möjligheten att ha BI-tillgångar lokalt med Power BI-rapportservern. Power BI-rapportservern är en lokal server som möjliggör distribution av interaktiva Power BI-rapporter och sidnumrerade rapporter helt inom gränserna för organisationens brandvägg. Med Power BI Premium kan samma antal virtuella kärnor som en organisation tillhandahåller i molnet också distribueras lokalt via Power BI-rapportservern utan att behöva dela kapaciteten. Organisationer kan välja Power BI i molnet, eller välja att behålla rapporter lokalt med Power BI-rapportservern och flytta till molnet i egen takt.

Mer information finns i [Kom igång med Power BI-rapportserver](report-server/get-started.md).

**Hur kan kunderna få åtkomst till Power BI-rapportservern?**  
Power BI-rapportservern är tillgänglig via Power BI Premium eller som en förmån för kunder med Microsoft SQL Server EE med SA. Kontakta din Microsoft-representant för mer information.

**Varför är inte Power BI-rapportservern inte tillgänglig som förmån för Microsoft SQL Server Standard- eller Enterprise Edition-kunder?**  
Power BI-rapportservern är tillgänglig för SQL Server-kunder med EE + SA-licenser eller för köp via Power BI Premium. Power BI är en SaaS-tjänst och Power BI-rapportservern följer samma prenumerationsbaserade leveransmodell.
